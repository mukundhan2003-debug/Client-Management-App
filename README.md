# ClientVault — Client Management System

A full-stack web app to securely manage client project records.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Flask (Python)
- **Database:** SQLite

## Setup

### Backend
```bash
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
npm install
npm start
```

## Features
- Secure signup & login with bcrypt password hashing
- Protected dashboard with client project records
- Professional dark UI with animated city background
